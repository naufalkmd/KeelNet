\begin{abstract}
\system\ studies a narrower and more operational version of hallucination control:
evidence-grounded question answering with an explicit answer-or-\abstain\
decision.
The current implementation focuses on a Stage 1 setting built on SQuAD~2.0,
where the model receives a question and a single evidence passage and must
either return an extractive answer span or abstain when support is insufficient.
This paper draft presents the motivation, task definition, experimental
protocol, and current modeling setup for that Stage 1 comparison between an
answer-only baseline and an abstain-aware variant.
Later stages are designed to extend this core setup with support verification,
confidence calibration, retrieval, and adaptive constraint balancing.
The results section is intentionally scaffolded so the final paper can be
updated as stable experimental findings arrive.
\end{abstract}
