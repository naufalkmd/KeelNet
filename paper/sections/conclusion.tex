This draft paper frames \system\ as a staged research program for
evidence-grounded question answering with controlled abstention.
The current implemented contribution is a narrow Stage 1 study that compares an
answer-only baseline with an abstain-aware alternative under a controlled
SQuAD~2.0 setting.

The main value of the current scaffold is that it lets the team write the paper
structure early while keeping the empirical claims honest.
As the experiments stabilize, this draft can be upgraded by filling in the
final results, adding error analysis, and tightening the discussion around
which later stages are actually justified by the Stage 1 evidence.
